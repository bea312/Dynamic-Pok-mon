# Dynamic Pokemon

A simple web app that fetches Pokemon data dynamically and displays it in the browser.

## Features

- Search or load Pokemon data dynamically
- Show key details like name, image, and stats
- Clean, responsive UI built with HTML, CSS, and JavaScript

## Tech Stack

- HTML
- CSS
- JavaScript
- PokeAPI

## Project Structure

- `index.html` - main page structure
- `script.js` - fetch logic and DOM updates
- `README.md` - project documentation

## Run Locally

1. Clone or download this repository.
2. Open `index.html` directly in your browser.

## How It Works

1. JavaScript sends a request to the Pokemon API.
2. The app receives Pokemon data in JSON format.
3. The UI updates dynamically with the returned data.
